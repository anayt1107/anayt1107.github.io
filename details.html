RGB Tesla Logo Sign

Date Created: November 16, 2022

This is a 3D printed Tesla logo which I modeled from a vector of the logo. I added channels for RGB lights from Amazon. I used it as wall decor for a while then sold it on Facebook Marketplace. By modeling this I learned many important skills in Blender, such as using a vector to create a 3D model, lighting in Blender, and printing an object in multiple parts.
